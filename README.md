# Packing Service

## Technology Stack:

* Java 17
* Gradle
* Spring Framework
* Spring Boot
* Spring MVC
* Spring Data Jpa
* Postgres
* Hibernate
* OpenApi (Swagger)

## How to Run:

* Under root directory run: `docker-compose up`
* Then run `./gradlew bootRun`

## API Documentation

After running project successfully go to
the : [http://localhost:8080/swagger-ui.html](http://localhost:8080/swagger-ui.html)
